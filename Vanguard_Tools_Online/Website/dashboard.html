Modul & Panduan Alat

Panduan teknis pengoperasian sistem monitoring Smart Chamber

Smart Chamber dirancang untuk mengambil sampel udara dan gas secara presisi. Sistem ini beroperasi menggunakan dua sensor utama untuk memastikan kualitas data:
  • Pemantauan Lingkungan: Sensor BME280 bekerja mengukur Suhu, Kelembapan, dan Tekanan Udara di sekitar area pengambilan sampel.
  • Pendeteksian Gas: Sensor MQ-4 secara aktif mendeteksi konsentrasi gas alam (Natural Gas) dan Metana (CH₄).
Seluruh proses pengambilan sampel dilakukan secara otomatis melalui mekanisme syringe, dan data sensor akan langsung dikirim serta disimpan ke dalam sistem web secara real-time.

Sistem dirancang berbasis cloud (tanpa penyimpanan SD Card lokal) sehingga data dapat diakses dari mana saja.
  • Login & Akses: Buka link web dashboard melalui browser di perangkat Anda, lalu masuk menggunakan kredensial yang telah diberikan.
  • Monitoring Data: Pada halaman utama (Beranda), Anda dapat langsung melihat pembaruan angka dari sensor BME280 dan MQ-4 yang disinkronisasi secara otomatis setiap kali alat mengambil sampel.

Alat ini terintegrasi dengan waktu online server (tanpa modul RTC fisik), sehingga jadwal yang Anda buat sangat akurat.
  • Pengaturan Jadwal: Navigasikan ke menu Otomatisasi/Jadwal di panel sebelah kiri.
  • Tentukan Waktu: Masukkan jam dan interval waktu yang diinginkan untuk pengambilan sampel. Sistem akan secara otomatis menggerakkan mekanisme syringe and merekam data sensor tepat pada waktu yang telah Anda tentukan.
